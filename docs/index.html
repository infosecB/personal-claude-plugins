Personal Claude Code Plugins

infosecB's personal marketplace of plugins for Claude Code and Cowork (Claude Desktop).

Marketplace Installation

Claude Code

  1. Open Claude Code
  2. Type /plugin and press Enter
  3. Tab to Marketplaces
  4. Select Add Marketplace
  5. Type infosecb/personal-claude-plugins and press Enter
  6. Select any plugins you want to enable
